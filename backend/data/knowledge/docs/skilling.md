# Skilling tradeoffs and method fit

Explains how to think about skilling methods beyond raw XP.

Good skilling advice should weigh raw XP, unlock value, profit profile, and attention cost. The strongest method is often the one the player can repeat consistently instead of the most theoretical XP method. Routine loops like Birdhouse runs can be high leverage because they compound passively without dominating a whole session.

Methods like High Alchemy, farming contracts, birdhouse runs, and Tempoross should also be judged by session shape. Some are better for low-attention routine value, while others are better for short focused bursts with stronger immediate engagement.

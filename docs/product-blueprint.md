# Cerebro Product Blueprint

## Product Name
Cerebro

## Overview
Cerebro is an AI-powered Old School RuneScape (OSRS) progression assistant that helps players make smarter decisions about training, quests, teleports, gear upgrades, and long-term account goals.

## Core Philosophy
Cerebro is NOT a bot and does NOT automate gameplay.

It is a decision-support system that:
- analyzes player progression
- recommends next actions
- explains game mechanics clearly
- helps players optimize their time

## Target Users
- New players
- Returning players
- Mid-game players
- Players pursuing specific goals (Fire Cape, Barrows Gloves, Quest Cape, etc.)

## Core Features (v1)

### 1. Ask Cerebro (AI Chat)
Users can ask:
- best training methods
- gear upgrades
- quest requirements
- teleport routes
- progression advice

### 2. Account Snapshot
Users can:
- enter RSN
- view skill levels
- get summary insights
- receive “next best action” suggestions

### 3. Skill Training Advisor
Inputs:
- skill
- level
- preference (fastest, afk, profitable)

Outputs:
- recommended methods
- requirements
- estimated XP rates
- explanation

### 4. Quest Helper
- requirements
- rewards
- importance of quest
- missing requirements based on player stats

### 5. Gear Advisor
- combat style
- budget
- current gear

Outputs:
- upgrade suggestions
- best value items
- progression path

### 6. Teleport Helper
- fastest routes
- requirement-based options
- alternatives

### 7. Goal Planner
Users can select goals:
- Fire Cape
- Barrows Gloves
- Quest Cape

System outputs:
- required skills
- required quests
- step-by-step plan

## Non-Goals
- No botting
- No automation
- No interaction with game client
- No violation of OSRS rules

## Success Criteria
- Accurate recommendations
- Clean modern UI
- Fast response times
- Useful progression insights

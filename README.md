# Cerebro OSRS

Cerebro is an AI-powered Old School RuneScape progression assistant designed to help players make smarter decisions about training, quests, teleports, gear upgrades, and account progression.

## Vision

Cerebro is not a bot or gameplay automation tool. It is a player advisory platform focused on:
- account-aware recommendations
- structured OSRS knowledge
- AI-assisted explanations
- modern RuneScape-themed UI
- production-minded full-stack architecture

## Planned Features

- AI chat assistant for OSRS help
- skill training recommendations
- teleport and travel helper
- quest requirement and reward guidance
- gear upgrade advisor
- goal planner
- account snapshot based on RSN

# Cerebro Frontend

Vite + React frontend for the `cerebro-osrs` monorepo.

## Environment

Create a frontend env file if you want to override the backend URL:

```bash
VITE_API_BASE_URL=http://127.0.0.1:8000/api
```

## Scripts

```bash
npm install
npm run dev
```

## First integrated surfaces

- dashboard
- account creation + sync
- next action recommendations
- goals + plan generation
- chat prompt wiring
- skills + quests pages
